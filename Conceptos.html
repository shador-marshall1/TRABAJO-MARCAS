Tags básicos


En todo documento de HTML SIEMPRE deben haber tags básicos que corresponden a todo documento que se pueda crear, hay 3 principales:
    1. HTML: Este será el principal, vendrá el principio y al final de todo documento

    2. HEAD: Aquí contendrá todos los metadatos posible y las referencias al link CSS, por ejemplo, que queramos crear.

    3. BODY: Como en todo documento de HTML, aquí irá el contenido de nuestra página, a continuación veremos como podremos referenciarlos en el CSS, usando algo llamado "div"
