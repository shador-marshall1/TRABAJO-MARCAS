Uso de tablas



¿Cómo crear y utilizar tablas?


Podemos utilizar muchas etiquetas, pero hay que tener un par de conceptos básicos en mente:

     1. Para crear una tabla se empieza con la etiqueta "Table", en la que dentro podremos especificar el borde de la misma
     2. Usaremos la etiqueta "TR" para crear una nueva fila, dentro de la misma vendrá la etiqueta "TD", para crear el contenido de cada fila
     Hay que tener en cuenta que dentro de TD podremos poner directrices usando otras etiquetas
     3. Etiqueta "COLSPAN" para elegir el número de filas que se expandirá, por ejemplo, "Recreo", tiene "COLSPAN="5""
     4. Etiqueta "rowspan" para seleccionar cuantas columnas hacia abajo ocupará, tener en cuenta que si tiene "rowspan="2"", el siguiente TR ocupará la correspondiente posicion
     5. Se puede usar "border" para seleccionar la anchura del borde de la tabla, la actual tiene "border=1px;", se puede usar "align" para seleccionar la pocision del texto, en este caso, RECREO tiene "ALING="CENTER""
     6. Etiqueta "Width", en CSS, para seleccionar el largo de página
     7. Dentro de los TD poremos poner le etiqueta "BGCOLOR="COLOR"" para seleccionar el color de fondo de las celdas, pondremos el fondo de color rojo todas las celdas que contengan FOL
